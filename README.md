## ABOUT THIS PROJECT

Simple Rest API Movies CRUD in c# using MongoDB Database.

## ENDPOINTS

Aplication Routes

```
  POST - Create a new movie
  PUT/:ID - Edit a movie
  DELETE - Remove a movie
  GET - List all movies
  GET/:ID - List a single movie

```

## STRUCTURE

Project Structure


```

 moviesapp
  
      - Models
      - Services
      - Controllers


```

## EXECUTING PROJECT

Command in terminal:

> dotnet run

https://localhost:5001/swagger/index.html
